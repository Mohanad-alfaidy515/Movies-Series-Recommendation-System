# Movies-Series-Recommendation-System
Movies &amp; Series Recommendation System built with FastAPI and MS SQL Server
CineMatch-Pro is a comprehensive Full-Stack web application designed to provide personalized movie and series recommendations. Built with a modern architecture, the system leverages FastAPI for a high-performance backend and Microsoft SQL Server for robust data management.
The platform features a sophisticated Role-Based Access Control (RBAC) system, distinguishing between Administrators (who manage the content library and user data) and Standard Users (who can browse, rate, and receive tailored suggestions). The frontend is crafted using Vanilla JavaScript and Bootstrap, ensuring a responsive and seamless user experience through asynchronous API integrations.
Key Features :
High-Performance API: Powered by FastAPI for rapid data processing.
Database Management: Integrated with MSSQL Server for secure and scalable storage.
RBAC System: Fully implemented Authentication & Authorization (Admin vs. User).
Dynamic UI: Responsive design with real-time content updates via Fetch API.
Scalable Architecture: Built with clean code principles, making it ready for future AI/Machine Learning model integration.
Tech Stack :
Backend: Python, FastAPI, SQLAlchemy.
Frontend: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5.
Database: Microsoft SQL Server.
Security: JWT (JSON Web Tokens) for secure sessions.
